搭建Node开发环境

Node.exe/Node.msi 软件的安装

1、直接双击安装某个版本
    不要安装中文目录下

2、nvm node version manager
    它可以让我们的电脑安装多个node版本，但是用的时候，只能用一个

    参考:
        http://www.jianshu.com/p/07c3456e875a

    nvm指令:
        nvm list 查看当前安装了哪些node版本
        nvm install 版本号 安装哪个版本
        nvm uninstall 版本号 卸载哪个版本
        nvm use 版本号，切换到哪个版本

如何验证有没有安装成功
    node -v 能出现版本号就代表成功了

设置系统环境变量

1、在终端的任何目录下，我都想启动QQ?配置环境变量

    步骤:
        1、拷贝我们可执行文件所在的目录 QQ.exe

        2、打开系统属性 > 高级系统设置 > 高级 > 环境变量 > 系统变量(一般配置这个即可)
            Path : C:\Program Files (x86)\Tencent\QQ\Bin

        3、重启cmd(必须)

        4、运行QQ.exe

2、系统环境变量的执行流程
    比如，在个人目录下，输入QQ
    1、会在当前个人目录下去找，有没有QQ.exe，如果
    有，直接打开，如果没有继续去系统环境变量里面找

    2、去系统环境变量中，一个一个去系统环境变量里
    面配置的路径，一个一个去找，如果找到就打开，如
    果没找到，就报"QQ不是内部或是外部指令"

系统环境变量类似于"桌面快捷方式"

REPL环境

REPL 交互式解释器
    交互式解释器（REPL）既可以作为一个独立的程序
    运行，也可以很容易地包含在其他程序中作为整体程
    序的一部分使用。REPL为运行JavaScript脚本与查
    看运行结果提供了一种交互方式，通常REPL交互方
    式可以用于调试、测试以及试验某种想法。

启动了一个应用程序之后，操作系统会给我们这个应用程序，分配以内存

开启:
    输入 node 启动了我们node.exe，开启了REPL环境

关闭:
    1、连按连个 Ctrl+C
    2、在终端中输入.exit

代码测试

1、把我们自己写的业务逻辑，放入一个单的js中

2、切换到我们项目所在的目录

3、直接使用 node 你要执行的js文件，就能看到运行效果了

注意点:
    当我们在终端里面输入 node 01.test.js，它其实内部做的事情是这样的
    1、开启REPL（启动node.exe）
    2、读取01.test.js中的内容，扔在REPL去解释执行，把结果输出到屏幕上
    3、自动退出REPL环境，释放资源

NodeJS对核心模块(http,url)

HTTP模块

站在服务端的角度，HTTP模块，主要是用来开启Web服务，供浏览器访问

服务端，返回的数据形式有很多，可能是一个普通的字符

代码的步骤说明

1、导入http模块
    需要使用到一个关键字，require('http')
    var http = require('http')

2、使用导入进来的http模块(对象)，创建一个服务
    var server = http.createServer()

3、监听浏览器请求，处理、响应
    server.on('request',function(request,response){
        response.end("Hello Girl!!!")
    })

4、启动
    server.listen(8899,"192.168.72.103",function(err){
        if(err){
            console.log(err)
        }

        console.log("start success!")
    })

注意点:
    如果我们更改了源代码，必须重新退出REPL，然后重新运行我们的代码，这样才能起作用

常见的问题:
    1、少写了端口号，或端口好的`:`写成了中文

URL模块

浏览器发送一个GET请求
    http://192.168.72.103:8899/login?username=zhangsan&pwd=123

后台要知道，浏览器传递过来的Path和参数

url模块，可以对'/login?username=zhangsan&pwd=123'，调用url模块里面的方法，就可以将这一长串字符串，把它转成js对象

代码步骤:
    1、var url = require('url')

    2、url.parse(URL字符串)

作用:
    将url字符串，转成js对象，然后，通过设置第二个参数为true，可以将键值对的参数，转成js对象，方便后续操作

nodemon

作用:
    它的作用就是用来监控我们源代码，发现源代码更
    改，保存之后，它就会先退出REPL环境，然后重新
    开启REPL，将最新的代码扔在REPL中去运行

nodemon是一个npm的全局包，安装方式如下
    npm i nodemon -g

使用
    nodemon 要执行文件名

注意点:
    如果用nodemon去监控源代码，发现源代码更改之后，它没有自动重启，尝试在终端里面按`一个`
    Ctrl+C

安装的方式不一样

let&const

定义变量/对象

let 就相当于var
const 定义常量，定义的东西不能改

实际开发中:
    1、一般情况下，当我们定义好了之后的值，不需要更改的时候，推荐使用const，常量效率高

    2、如果你定义的值，如果以后要改，就必须使用let定义

let 和 var不一样的地方
    var 允许变量提升
    let 不允许这样做
